# Linux Scripts

Bash scripts built during Phase 1 of my DevOps learning path.

## File Organizer (organizer2.sh)

Takes a folder path as argument, lists and counts files, moves all .txt files 
into a text/ subfolder, and logs every action with a timestamp to organizer.log.

### Usage
./organizer2.sh /path/to/folder

### What I learned
- Bash scripting basics
- Variables and command substitution
- Logging with timestamps
- File manipulation with mv and mkdir
