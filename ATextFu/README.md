# Linux Scripts
Bash scripts built during Phase 1 of my DevOps learning path.

## Scripts

### TextFu/organizer2.sh
Organizes files in a folder — moves .txt files to a subfolder and logs actions.
Usage: ./organizer2.sh /path/to/folder

### ATextFu/log-analyzer.sh
Analyzes a log file — counts INFO, WARNING, and ERROR lines, extracts error messages, saves report to log-report.txt.
Usage: ./log-analyzer.sh <logfile>
